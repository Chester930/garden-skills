# 產品精修工作流模板

本文件用於「在原圖基礎上對產品做精修」的編輯任務：

- 提升商品圖質感
- 修補瑕疵 / 劃痕
- 調整光澤 / 反光
- 美化標籤 / 銳化文字
- 重新整理陰影 / 倒影

特徵：

- 主體（產品）保留
- 重點是「質感升級」而不是「替換」
- 不強行換背景
- 不修改產品形態與文字

## 適用範圍

- 商品圖質感升級
- 瑕疵修復
- 燈光重做
- 標籤 / 文字銳化

## 何時使用

- 用戶提供原圖（REFERENCE_0）+ 希望產品更高級
- 用戶希望保留場景，僅升級質感

不要使用：

- 整張換背景（用 `background-replacement.md`）
- 替換產品爲另一產品（用 `local-object-replacement.md`）
- 把白底主圖重新生成（用 `product-visuals/white-background-product.md` 重畫）

## 缺失信息優先提問順序

1. 原產品描述
2. 想強化的方面（質感 / 光澤 / 標籤 / 陰影）
3. 是否要去掉瑕疵
4. 是否保留背景
5. 是否雙圖對比輸出

## 主模板：產品質感升級

📖 描述

保留原圖的產品與場景，只對產品本身做質感升級（更通透 / 更有光澤 / 標籤更清晰 / 更高級）。

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留畫面中 {argument name="product" default="白色按壓瓶"} 的形狀、比例、標籤內容與場景背景，對產品本身做質感升級：
- 提升 {argument name="enhancement focus" default="瓶身光澤與微反光"} 的質感；
- 修復 {argument name="defect to fix" default="瓶身上輕微的劃痕與髒點"}；
- 讓 {argument name="label sharpening" default="正面標籤的字與 logo"} 更銳利清晰；
- 陰影與反光與原圖保持方向一致，但更細膩；
- 不要修改產品顏色、文字、材質類型；
- 不要替換背景。
渲染風格：{argument name="render style" default="高端商業產品攝影 + 雜誌級後期"}。
```

### 參數策略

- 必問：要強化的方面
- 可默認：保留背景、保留文字、保留色彩
- 可隨機：陰影柔和度

### 自動補全策略

- 默認提升光澤 + 修復瑕疵 + 銳化標籤
- 不指定強度時按「克制升級」處理
- 不動品牌色

## 變體 1：標籤 / 文字銳化

📝 提示詞

```text
以 REFERENCE_0 爲基礎，僅對畫面中產品標籤上的文字做銳化與清晰化處理：
- 保留所有文字內容、字距、字體；
- 不修改瓶身其他部分；
- 不修改背景。
```

## 變體 2：陰影 / 倒影重做

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留產品本身，重新生成更精緻的 {argument name="shadow type" default="底部柔光陰影 + 微反光"}：
- 陰影方向與原圖燈光一致；
- 反光強度克制不搶鏡；
- 不修改產品本身；
- 不修改背景。
```

## 變體 3：自動補全模式

📝 提示詞

```text
以 REFERENCE_0 爲基礎，對產品做整體質感升級，自動決定升級重點（光澤 / 標籤 / 陰影 / 邊緣）；
保持產品原本身份，整體提升 1 個檔次。
```

## 避免事項

- 不要修改產品的文字（特別是品牌名）
- 不要讓光澤提升變成「塑料感」
- 不要重新生成背景（除非用戶允許）
- 不要讓陰影方向漂移
- 不要對產品做誇張色彩調整
