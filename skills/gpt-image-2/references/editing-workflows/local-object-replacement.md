# 局部對象替換工作流模板

本文件用於「將原圖中某一對象替換爲另一對象」的編輯任務：

- 把圖中咖啡杯換成保溫杯
- 把圖中人物 T 恤換成衛衣
- 把車 logo 換爲另一品牌
- 把寵物從貓換成狗
- 把背景裏某物換爲另一物

特徵：

- 主體大部分保留
- 僅局部精確替換
- 周圍光線 / 陰影需要適配
- 必要時配合 mask（蒙版）

## 適用範圍

- 單一對象替換
- 多對象替換
- 配合蒙版的精確替換

## 何時使用

- 用戶提供原圖（REFERENCE_0）+ 想換某物
- 用戶希望除被替換對象外其他都不動

不要使用：

- 整張換背景（用 `background-replacement.md`）
- 僅刪除某物（用 `object-removal.md`）
- 產品 / 人像精修（用 `product-retouching.md` / `portrait-local-edit.md`）

## 缺失信息優先提問順序

1. 原圖中要替換的對象
2. 要替換爲什麼
3. 是否提供蒙版
4. 替換後是否需要重新陰影 / 反光
5. 是否保留替換對象的尺寸 / 位置

## 主模板：單對象替換

📖 描述

精確替換一個對象，其餘畫面儘量保留。

📝 提示詞

```text
以 REFERENCE_0 爲基礎，將 {argument name="original object" default="桌上的白色陶瓷咖啡杯"} 替換爲 {argument name="replacement object" default="同尺寸的不鏽鋼保溫杯，啞光銀色，瓶身有簡潔品牌字 'AURORA'"}。
保留原圖中其他所有元素的位置、光線、陰影與構圖；只對被替換對象本身做修改。
爲新對象重新生成與原圖光線方向一致的陰影、反光與材質。
不要改變其它人物、桌面、背景。
```

### 參數策略

- 必問：原對象、替換對象
- 可默認：是否需要重新陰影
- 可隨機：替換對象的次要細節

### 自動補全策略

- 默認保留原對象尺寸與位置
- 默認重新生成陰影
- 用戶沒指定材質時，按合理類比選

## 變體 1：配合蒙版的精確替換

📝 提示詞

```text
以 REFERENCE_0 爲基礎，使用 REFERENCE_1（蒙版）所標記的區域，精確替換 {argument name="object to replace" default="人物的白色 T 恤"} 爲 {argument name="new object" default="深藍色長袖衛衣，胸前印有 'AURORA' 字樣"}。
僅對蒙版區域做修改，其餘區域必須像素級保留；
爲新衣服生成與原圖燈光一致的褶皺與陰影；
保持人物身材與姿勢完全不變。
```

## 變體 2：批量對象替換

📝 提示詞

```text
以 REFERENCE_0 爲基礎，將畫面中所有 {argument name="original objects" default="木質椅子"} 替換爲 {argument name="replacement objects" default="米色塑膠椅"}。
保持每把椅子的位置、角度與擺放不變；
爲新椅子生成與原圖光線方向一致的陰影；
不要修改桌子、牆面、燈具、人物。
```

## 變體 3：自動補全模式

📝 提示詞

```text
以 REFERENCE_0 爲基礎，將原圖中 {argument name="object" default="主要前景物體"} 替換爲視覺風格更現代的同功能版本，自動決定材質與配色，但保持位置與尺寸一致。
```

## 避免事項

- 不要替換後改變原對象的位置 / 比例（除非用戶允許）
- 不要讓替換對象的燈光方向與原圖不一致
- 不要順便修改其它無關元素
- 不要讓替換對象的材質顯得"貼上去"
- 沒有蒙版時，不要假裝精確（說明邊緣可能略微變化）
