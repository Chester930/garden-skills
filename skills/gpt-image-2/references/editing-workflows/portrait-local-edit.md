# 人像局部修改工作流模板

本文件用於「對人像圖做局部修改」的編輯任務：

- 修改髮型 / 發色
- 修改服裝顏色 / 款式
- 修改眼鏡 / 配飾
- 修改妝容
- 修改表情（輕度）

特徵：

- 必須保留人物身份（臉型 / 五官比例）
- 僅局部修改
- 必要時配合蒙版
- 不修改場景

## 適用範圍

- 人像局部修改
- 服裝 / 配飾修改
- 妝容修改
- 髮型 / 發色修改

## 何時使用

- 用戶提供人像圖 + 想修改某一局部
- 用戶希望保留「是同一個人」的感覺

不要使用：

- 整張換背景（用 `background-replacement.md`）
- 替換爲另一對象 / 物品（用 `local-object-replacement.md`）
- 刪除某物（用 `object-removal.md`）

## 缺失信息優先提問順序

1. 想修改的部位（髮型 / 服裝 / 妝容 / 配飾）
2. 修改後的樣子描述
3. 是否提供蒙版
4. 是否保留表情 / 姿勢
5. 是否調整光線

## 主模板：髮型 / 發色修改

📖 描述

保留人物身份與構圖，僅修改髮型 / 發色。

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留人物的臉型、五官比例、膚色、表情、姿勢、服裝與背景，僅修改髮型 / 發色爲：
{argument name="new hair description" default="齊肩短髮 + 自然黑色 + 輕微微卷"}。
新髮型的陰影、反光、髮際線必須與原圖燈光方向一致；
不要修改人物身份特徵（眼睛形狀、嘴形、鼻子、耳朵）；
不要修改背景。
```

### 參數策略

- 必問：修改部位、新樣子
- 可默認：保留身份、保留背景、保留燈光
- 可隨機：髮絲細節

### 自動補全策略

- 默認保留人物身份
- 默認保留構圖與燈光
- 默認僅做局部修改，不動其它

## 變體 1：服裝修改

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留人物的臉、髮型、姿勢、表情與背景，僅修改服裝爲：
{argument name="new outfit description" default="米色長袖襯衫 + 卡其色西裝外套"}；
新服裝的褶皺、陰影必須與原圖燈光方向一致；
不要修改人物身份特徵；
不要修改背景。
```

## 變體 2：妝容修改

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留人物身份與構圖，僅修改妝容爲：
{argument name="new makeup description" default="啞光裸色脣 + 棕調眼影 + 自然腮紅"}；
保留膚色、膚質細節、五官形狀；
不要讓妝容顯得過濃或與膚色不匹配；
不要修改背景。
```

## 變體 3：配飾修改 / 添加

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留人物身份與構圖，添加 / 修改配飾爲：
{argument name="accessory description" default="一副細金屬邊圓框眼鏡"}；
配飾的角度、陰影必須與原圖燈光方向一致；
不要修改其它身體部位；
不要修改背景。
```

## 變體 4：自動補全模式

📝 提示詞

```text
以 REFERENCE_0 爲基礎，對人物做輕度風格升級（髮型 / 妝容 / 服裝 任一），自動判斷當前最不協調處並修復；
保持人物身份不變；
保持背景不變。
```

## 避免事項

- 不要修改人物五官導致換臉
- 不要讓修改後的局部光線方向與原圖不一致
- 不要順帶修改背景與膚色
- 不要讓妝容濃到「濾鏡假皮膚」
- 沒有蒙版時，告訴用戶結果可能略有偏差
- 不要在用戶沒要求時修改性別 / 種族特徵
