# 背景替換工作流模板

本文件用於「將一張圖的背景替換爲新背景」的編輯任務，對應腳本 `scripts/edit.js`。

適合場景：

- 商品圖換背景（白底 → 生活場景 / 影棚 / 戶外）
- 人像換背景（雜亂 → 乾淨棚景）
- 老照片背景翻新
- 跨品類素材統一背景風格

## 適用範圍

- 單主體圖換背景
- 主體不變 / 僅背景變
- 主體邊緣清晰 / 可識別

## 何時使用

- 用戶提供原圖（REFERENCE_0）+ 一句「換成 XX 背景」
- 用戶希望主體不動只換背景
- 用戶希望多張圖統一背景

不要使用：

- 主體本身需要修改（用 `local-object-replacement.md`）
- 僅去除某物（用 `object-removal.md`）
- 產品本身需要精修（用 `product-retouching.md`）

## 缺失信息優先提問順序

1. 原圖描述 / 主體是什麼
2. 新背景：場景 / 色調 / 燈光
3. 是否保留原圖燈光方向
4. 是否需要重新陰影 / 反光
5. 輸出比例（保持原圖 / 調整）

## 主模板：商品圖換背景

📖 描述

輸入一張主體圖，輸出主體保留、背景換成指定場景的圖。

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留 {argument name="subject" default="畫面中央的白色按壓瓶"} 的形態、比例、標籤和材質，僅將背景替換爲 {argument name="new background" default="清晨陽光下的木質梳妝檯，柔光從左側窗戶灑入，遠景輕微虛化，背景元素包含一杯水、幾片白色花瓣、摺疊的米色毛巾"}。
重新生成與新背景一致的陰影與反光，讓主體看起來真實地存在於新場景中。
不要修改主體本身的顏色、文字、形狀或材質。
渲染風格：{argument name="render style" default="高分辨率商業攝影，顆粒感真實，淺景深，主體清晰，背景自然虛化"}。
輸出比例：{argument name="aspect ratio" default="保持原圖比例"}。
```

### 參數策略

- 必問：原圖主體、新背景描述
- 可默認：渲染風格、比例
- 可隨機：背景細節小道具

### 自動補全策略

- 行業自動選背景：化妝品 → 梳妝檯；食品 → 餐桌；電子 → 極簡辦公桌
- 默認保持原圖比例
- 默認重新生成陰影

## 變體 1：人像換棚景

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留 {argument name="subject" default="畫面中的人物"} 的姿勢、表情、穿着與五官，僅將背景替換爲 {argument name="studio backdrop" default="中性灰背景紙"}。
重新生成與新背景一致的柔光陰影；不要改變人物形象、膚色、服裝顏色。
渲染風格：{argument name="render style" default="棚拍人像攝影，柔光，自然膚質"}。
```

## 變體 2：商品圖換戶外場景

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留 {argument name="subject" default="畫面中央的產品"}，將背景替換爲 {argument name="outdoor scene" default="海邊木棧道，黃昏暖光，遠處海浪虛化"}。
保留產品所有標籤與材質細節；爲產品重新生成與戶外光線方向一致的陰影。
不要讓產品顏色因光線偏移過強（保持品牌色）。
```

## 變體 3：自動補全模式

📝 提示詞

```text
以 REFERENCE_0 爲基礎，保留主體；自動選擇最適合該主體的「乾淨影棚 / 自然場景 / 極簡室內」三種背景之一併替換。
保持原圖比例；爲主體重新生成自然陰影。
```

## 避免事項

- 不要修改主體本身（除非用戶允許）
- 不要讓光線方向與主體原本受光不一致
- 不要讓背景元素太多分散注意力
- 不要讓主體邊緣出現明顯摳圖痕跡
- 不要修改主體上的文字 / 標籤
