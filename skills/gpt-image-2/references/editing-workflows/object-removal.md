# 雜物去除工作流模板

本文件用於「去除圖中某物，保留其餘畫面」的編輯任務：

- 去除路人 / 圍觀者
- 去除水印 / logo（僅限非版權場景）
- 去除多餘道具
- 去除雜線 / 電線
- 去除畫面中的瑕疵

特徵：

- 不需要替換爲新對象
- 移除區域需自然填充
- 必要時配合蒙版
- 不能影響主體

## 適用範圍

- 單雜物去除
- 多雜物去除
- 大面積去除（電線 / 線纜）
- 局部瑕疵去除

## 何時使用

- 用戶提供原圖（REFERENCE_0）+ 想去掉某物
- 用戶希望畫面更乾淨

不要使用：

- 替換爲新對象（用 `local-object-replacement.md`）
- 整張換背景（用 `background-replacement.md`）

## 缺失信息優先提問順序

1. 要去除的對象
2. 是否提供蒙版
3. 去除後該區域應填充爲什麼（背景延續 / 乾淨底色）
4. 是否影響陰影 / 反光

## 主模板：單雜物去除

📖 描述

精確去除一個對象，原區域用周圍環境自然填充。

📝 提示詞

```text
以 REFERENCE_0 爲基礎，去除畫面中的 {argument name="object to remove" default="背景裏的路人"}。
被去除區域使用周圍環境（{argument name="fill description" default="街道地面、牆面與遠景建築"}）做自然延續，使其看起來從未存在；
保留主體、光線方向、陰影與畫面構圖；
不要修改其它人物、物體或文字。
```

### 參數策略

- 必問：要去除的對象、填充類型
- 可默認：保留主體
- 可隨機：補全細節

### 自動補全策略

- 默認根據周圍環境推斷填充
- 默認保留所有陰影方向

## 變體 1：配合蒙版去除

📝 提示詞

```text
以 REFERENCE_0 爲基礎，使用 REFERENCE_1（蒙版）所標記的區域作爲待去除區域，將該區域內容自然替換爲周圍環境的延續；
僅對蒙版區域做修改，蒙版外像素級保留；
不要讓填充區域出現可識別的接縫或紋理跳變。
```

## 變體 2：批量去除（如電線 / 圍觀者）

📝 提示詞

```text
以 REFERENCE_0 爲基礎，去除畫面中所有 {argument name="objects to remove" default="天空中的電線 / 街道上的路人"}。
被去除區域用對應背景（天空 / 地面 / 建築）自然延續；
保留主體、燈光、陰影、構圖。
```

## 變體 3：自動補全模式

📝 提示詞

```text
以 REFERENCE_0 爲基礎，自動識別畫面中幹擾主體閱讀的雜物並去除（如：路人、電線、瓶罐、水漬）；
保留主體與構圖；自然填充背景。
```

## 避免事項

- 不要把主體一併去掉
- 不要修改光線方向（去除後陰影也要保留）
- 不要讓填充區域出現「糊感 / 重複紋理」
- 沒有蒙版時，告訴用戶僅能盡力（結果可能有輕微差異）
- 不要去除帶版權 logo / 水印（除非用戶爲合法所有人）
